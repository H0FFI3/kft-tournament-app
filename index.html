Google Sheets Verbindung

Füge hier die ID deiner Google Sheets Tabelle ein. Die ID findest du in der URL:
docs.google.com/spreadsheets/d/DIESE-ID-HIER/edit

Google Sheets einrichten

Schritt 1: Erstelle eine neue Google Sheets Tabelle.

Schritt 2: Erstelle diese Tabellenblätter (genau so benannt):
Spiele  |  Mannschaften

Schritt 3 – Tabellenblatt "Spiele" Spalten:
A: SpielID B: Runde C: Gruppe D: Platz E: Datum F: Uhrzeit G: Team1 H: Team2 I: Tore1 J: Tore2 K: Status

Spalte K (Status): geplant, läuft oder beendet
Spalte D (Platz): Platz 1 oder Platz 2
Spalte B (Runde): Gruppe, Achtelfinale, Viertelfinale, Halbfinale, Finale
Spalte C (Gruppe): A, B, C, D, E oder F (nur für Gruppenspiele)

Schritt 4 – Tabellenblatt "Mannschaften" Spalten:
A: Name B: Gruppe C: Emoji/Flagge

Schritt 5: Klicke auf Freigeben → Jeder mit dem Link → Betrachter

Schritt 6: Aktiviere die Google Sheets API für deine Tabelle unter:
Erweiterungen → Apps Script oder nutze den einfacheren CSV-Export-Trick weiter unten.

Tipp (einfachster Weg): Nutze den CSV-Export ohne API-Key. Dazu muss die Tabelle öffentlich (nur Lesen) geteilt sein. Die App lädt dann automatisch alle 60 Sekunden neu.
